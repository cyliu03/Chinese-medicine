# 岐黄AI 智能中医诊断系统 - 部署指南

## 📋 火速部署（5分钟）

### Windows用户
```cmd
# 1. 安装依赖
cd server
pip install -r requirements.txt
cd ..
npm install

# 2. 启动服务
start.bat
```

### Mac/Linux用户
```bash
# 1. 安装依赖
cd server && pip install -r requirements.txt
cd .. && npm install

# 2. 启动服务
./start.sh
```

## 🔧 配置修改

### 修改后端端口
编辑 `server/main.py`:
```python
uvicorn.run(app, host="0.0.0.0", port=8000)  # 默认8000端口
```

### 修改前端API地址
编辑 `src/lib/aiEngine.js`:
```javascript
const API_URL = 'http://localhost:8000';  # 后端API地址
```

## 🧪 测试流程

### 1. 患者端测试
1. 访问 http://localhost:3000/patient
2. 填写患者信息
3. 完成四诊（望闻问切）
4. 查看AI诊断结果
5. 提交诊断

6. 在医生端查看
7. 审核并批准处方

### 2. 医生端测试
1. 访问 http://localhost:3000/doctor
2. 查看诊断列表
3. 点击诊断查看详情
4. 审核AI推荐处方
5. 查看相似病情名医药方
6. 添加医生批注
7. 批准或驳回处方

8. 查看药方知识库
9. 搜索药方
10. 添加自定义药方

## ❓ 壸见问题

### Q: 端口被占用
```
# Windows查看端口占用
netstat -ano | findstr :8000
netstat -ano | findstr :3000

# 修改端口
# 后端: 修改 server/main.py
# 前端: 修改 package.json 中的 port
```

### Q: AI诊断一直转圈
```
# 检查后端是否启动
curl http://localhost:8000/docs

# 检查前端API配置
# 确保 src/lib/aiEngine.js 中的 API_URL 正确
```
### Q: 模型加载失败
```
# 检查模型文件是否存在
dir training\checkpoints\best_model.pt

# 检查数据文件是否存在
dir training\data\chatmed\meta.json
```
