# 岐黄AI 快速在线部署

## 🎯 最简单的方法：使用ngrok（5分钟完成）

### 第一步：下载ngrok
1. 访问 https://ngrok.com/download
2. 下载Windows版本
3. 解压到任意目录
4. 注册ngrok账号（免费）

### 第二步：启动后端
```cmd
cd e:\Chinese-medicine\server
python main.py
```

### 第三步：暴露后端（新终端）
```cmd
ngrok http 8000
```
复制ngrok提供的地址，例如：`https://xxxx.ngrok.io`

### 第四步：修改前端配置
编辑 `src/lib/aiEngine.js`，将API_URL改为ngrok地址：
```javascript
const API_URL = 'https://xxxx.ngrok.io';  // 替换为你的ngrok地址
```

### 第五步：启动前端
```cmd
npm run dev
```

### 第六步：暴露前端（新终端）
```cmd
ngrok http 3000
```

### 完成！ 
将ngrok提供的前端地址发送给医生即可访问。

## 🌐 方案二：Render + Vercel（永久部署）

### 后端部署到Render
1. 访问 https://render.com
2. 注册账号
3. 创建新的Web Service
4. 连接GitHub仓库
5. 配置：
   - Build Command: `cd server && pip install -r requirements.txt`
   - Start Command: `cd server && python main.py`
   - 环境变量: `PYTHON_VERSION=3.10`

### 前端部署到Vercel
1. 访问 https://vercel.com
2. 导入GitHub仓库
3. 环境变量配置:
   - `NEXT_PUBLIC_MODEL_API_URL` = 你的Render后端地址

1. 申请免费云服务器
2. 安装Node.js和Python
