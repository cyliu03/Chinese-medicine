## 📤 上传到GitHub

### 方式一：使用PowerShell脚本（推荐）

```powershell
cd e:\Chinese-medicine
.\upload.ps1
```

按提示输入GitHub用户名即可。

### 方式二：手动上传

```bash
# 1. 初始化Git仓库
cd e:\Chinese-medicine
git init
git branch -M main

# 2. 添加远程仓库
git remote add origin https://github.com/你的用户名/qihuang-ai.git

# 3. 添加文件
git add .

# 4. 提交
git commit -m "Initial commit: 岐黄AI智能中医诊断系统"

# 5. 推送
git push -u origin main
```

## 🤗 上传模型到HuggingFace

### 步骤1：登录HuggingFace

```bash
# 登录HuggingFace
huggingface-cli login

# 输入Access Token
# 获取Token: https://huggingface.co/settings/tokens
```

### 步骤2：创建模型仓库

```bash
# 创建模型仓库
huggingface-cli repo create qihuang-ai-model --type model
```

### 步骤3：上传模型文件

```bash
# 进入模型目录
cd e:\Chinese-medicine\training\checkpoints

# 初始化Git
git init
git branch -M main

# 添加远程仓库
git remote add origin https://huggingface.co/你的用户名/qihuang-ai-model

# 添加文件
git add best_model.pt README.md

# 提交
git commit -m "Upload LSAN model for TCM diagnosis"

# 推送
git push -u origin main
```

### 步骤4：上传数据文件

```bash
# 进入数据目录
cd e:\Chinese-medicine\data\chatmed

# 添加远程仓库（如果还没添加）
git remote add origin https://huggingface.co/你的用户名/qihuang-ai-model

# 添加文件
git add meta.json

# 提交并推送
git commit -m "Add vocabulary data"
git push
```

## 🔧 常见问题

### Q: Git push失败
**A: 使用Personal Access Token**
1. 访问 https://github.com/settings/tokens
2. 创建新的Token（勾选repo权限）
3. 推送时用Token代替密码

### Q: 大文件上传失败
**A: 使用Git LFS**
```bash
git lfs install
git lfs track "*.pt"
git lfs track "*.json"
git add .gitattributes
git commit -m "Enable Git LFS"
```

### Q: HuggingFace上传失败
**A: 检查登录状态**
```bash
huggingface-cli whoami
# 如果未登录，运行：
huggingface-cli login
```
