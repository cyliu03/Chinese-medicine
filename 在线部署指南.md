# 岐黄AI 在线部署指南

## 🌐 部署方案选择

由于Vercel在中国大陆访问不稳定，推荐以下方案：

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **Railway** | 支持Python后端、免费额度大 | 需要GitHub账号 | ⭐⭐⭐⭐⭐ |
| **Render** | 完全免费、简单 | 冷启动较慢 | ⭐⭐⭐⭐ |
| **Zeabur** | 国内访问快、一键部署 | 免费额度有限 | ⭐⭐⭐⭐ |
| **阿里云/腾讯云** | 国内访问最快 | 需要付费 | ⭐⭐⭐ |

---

## 🚀 方案一：Railway部署（推荐）

### 步骤1：准备GitHub仓库
```bash
# 1. 在GitHub创建新仓库
# 2. 推送代码到GitHub
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/qihuang-ai.git
git push -u origin main
```

### 步骤2：部署后端到Railway
1. 访问 https://railway.app
2. 使用GitHub账号登录
3. 点击 "New Project"
4. 选择 "Deploy from GitHub repo"
5. 选择你的仓库
6. 添加环境变量：
   - `PYTHON_VERSION=3.10`
   - `PORT=8000`

### 步骤3：部署前端到Vercel
1. 访问 https://vercel.com
2. 导入GitHub仓库
3. 添加环境变量：
   - `NEXT_PUBLIC_MODEL_API_URL` = 你的Railway后端地址

---

## 🚀 方案二：Render部署（完全免费）

### 后端部署
1. 访问 https://render.com
2. 创建新的Web Service
3. 连接GitHub仓库
4. 配置：
   - Environment: Python 3
   - Build Command: `pip install -r server/requirements.txt`
   - Start Command: `cd server && python main.py`
   - Instance Type: Free

### 前端部署
1. 创建新的Static Site
2. 连接GitHub仓库
3. 配置：
   - Build Command: `npm run build`
   - Publish Directory: `.next`
   - 环境变量: `NEXT_PUBLIC_MODEL_API_URL` = 你的Render后端地址

---

## 🚀 方案三：Zeabur部署（国内访问快）

1. 访问 https://zeabur.com
2. 使用GitHub登录
3. 创建新项目
4. 添加服务：
   - 前端: 选择Next.js
   - 后端: 选择Python
5. 自动部署

---

## 🚀 方案四：使用ngrok临时部署（测试用）

### 安装ngrok
```bash
# Windows (使用scoop)
scoop install ngrok

# Mac
brew install ngrok

# Linux
snap install ngrok
```

### 启动后端并暴露
```bash
# 1. 启动后端
cd server
python main.py

# 2. 新终端，暴露后端
ngrok http 8000
```

### 启动前端并暴露
```bash
# 1. 修改前端API地址
# 编辑 src/lib/aiEngine.js
# const API_URL = 'https://xxx.ngrok.io';  # 替换为ngrok地址

# 2. 启动前端
npm run dev

# 3. 新终端，暴露前端
ngrok http 3000
```

### 获取公网地址
ngrok会提供类似 `https://xxx.ngrok.io` 的地址，可以直接访问。

## 📝 部署后配置

### 修改前端API地址
编辑 `src/lib/aiEngine.js`:
```javascript
// 将localhost替换为你的后端地址
const API_URL = process.env.NEXT_PUBLIC_MODEL_API_URL || 'https://你的后端地址';
```

### 添加环境变量
在部署平台添加：
- `NEXT_PUBLIC_MODEL_API_URL` = 后端完整地址

## 🔧 常见问题

### Q: 跨域问题
后端添加CORS配置:
```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
)
```

### Q: 模型文件太大
使用Git LFS:
```bash
git lfs track "*.pt"
git lfs push --all origin main
```

### Q: 冷启动慢
Render免费版会休眠，首次访问需要等待30秒左右唤醒。
