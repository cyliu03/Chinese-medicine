## 📋 上传前准备

### 1. 获取GitHub Token
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 点击 "Generate token"
5. **复制Token**（只显示一次）

### 2. 获取HuggingFace Token
1. 访问 https://huggingface.co/settings/tokens
2. 点击 "New token"
3. 选择 "Write" 权限
4. 点击 "Generate token"
5. **复制Token**

## 🔧 常见问题

### Q: Git push失败
**A: 使用Token代替密码**
- 用户名：你的GitHub用户名
- 密码：GitHub Personal Access Token

### Q: HuggingFace上传失败
**A: 检查Token权限**
- 确保Token有 "Write" 权限

### Q: 文件太大
**A: 模型文件已在.gitignore中排除**
- 模型文件需要单独上传到HuggingFace
