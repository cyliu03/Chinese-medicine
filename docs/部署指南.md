# 岐黄AI 智能中医诊断系统 - 部署指南

## 📋 禂述

本系统是一个基于深度学习的智能中医诊断系统，包含患者端和医生端两个界面。

### 系统特点
- 🏥 患者端：智能问诊、四诊采集、AI诊断
- 👨‍⚕️ 医生端：诊断审核、药方知识库、相似病情推荐
- 🤖 AI引擎：推荐系统 + 深度学习模型并行融合

## 🚀 快速部署

### 方式一：一键启动（推荐）

**Windows用户：**
```bash
# 双击运行 start.bat
start.bat
```

**Mac/Linux用户:**
```bash
chmod +x start.sh
./start.sh
```

### 方式二：手动启动

**1. 启动后端服务**
```bash
cd server
pip install -r requirements.txt
python main.py
```
后端服务将运行在 http://localhost:8000

**2. 启动前端服务**
```bash
npm install
npm run dev
```
前端服务将运行在 http://localhost:3000

## ❓ 常见问题

### Q: 后端启动失败
**A: 检查Python版本和依赖**
```bash
python --version
pip install -r server/requirements.txt
```

### Q: 前端启动失败
**A: 检查Node.js版本和依赖**
```bash
node --version
npm install
```

### Q: AI诊断一直转圈
**A: 检查后端服务是否正常运行**
- 确认后端在8000端口运行
- 检查前端API配置是否正确

### Q: 端口被占用
**A: 修改端口**
- 后端: 修改 server/main.py 中的端口
- 前端: 修改 src/lib/aiEngine.js 中的 API_URL
