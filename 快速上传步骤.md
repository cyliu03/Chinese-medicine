# 🚀 快速上传步骤

## 第一步：在GitHub创建仓库

1. 访问 https://github.com/new
2. 仓库名称填写: `qihuang-ai`
3. 选择 Public
4. **不要**勾选 "Add a README file"
5. 点击 "Create repository"

## 第二步：推送代码到GitHub

打开PowerShell，运行以下命令：

```powershell
cd e:\Chinese-medicine

# 设置分支
git branch -M main

# 添加远程仓库（替换为你的用户名）
git remote add origin https://github.com/你的用户名/qihuang-ai.git

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 岐黄AI智能中医诊断系统"

# 推送（会提示输入用户名和密码）
git push -u origin main
```

**注意**: GitHub现在需要使用Personal Access Token代替密码
- 创建Token: https://github.com/settings/tokens
- 勾选 `repo` 权限
- 推送时用Token代替密码

## 第三步：在HuggingFace创建模型仓库

1. 访问 https://huggingface.co/new
2. 选择 "Model"
3. 仓库名称填写: `qihuang-ai-model`
4. 选择 "MIT" 许可证
5. 点击 "Create model"

## 第四步：上传模型到HuggingFace

```powershell
# 登录HuggingFace（需要先获取Token）
huggingface-cli login
# Token获取地址: https://huggingface.co/settings/tokens

# 进入模型目录
cd e:\Chinese-medicine\training\checkpoints

# 初始化Git
git init
git branch -M main

# 添加远程仓库（替换为你的用户名）
git remote add origin https://huggingface.co/你的用户名/qihuang-ai-model

# 添加文件
git add best_model.pt README.md

# 提交
git commit -m "Upload LSAN model for TCM diagnosis"

# 推送
git push -u origin main
```

## 第五步：上传数据文件到HuggingFace

```powershell
# 进入数据目录
cd e:\Chinese-medicine\data\chatmed

# 初始化Git
git init
git branch -M main

# 添加远程仓库（替换为你的用户名）
git remote add origin https://huggingface.co/你的用户名/qihuang-ai-model

# 添加文件
git add meta.json

# 提交
git commit -m "Add vocabulary data"

# 推送
git push
```

## ✅ 完成！

上传完成后：
- GitHub代码仓库: `https://github.com/你的用户名/qihuang-ai`
- HuggingFace模型: `https://huggingface.co/你的用户名/qihuang-ai-model`
